Proyectos

Tienda-v

Este es un proyecto personal donde pongo en práctica los conocimientos que adquirí en el curso de NodeJs con Express. Es un Api Rest que cuenta con tres categorías pensados en la gestión de una tienda, además este servidor soporta los cuatro métodos principales del protocolo HTTP.

Lista de Juegos

Página web que desarrolle enfocado a los videojuegos, donde podemos ver una lista de videojuegos, en cada apartado se puede ver una imagen y una breve descripción del mismo indicando fecha de lanzamiento, plataforma y año de estreno entre otros. Fue desarrollado con Vainilla Javascript, Bootstrap 5 para el maquetado y HTML5.

Invasión Zombie

Es un videojuego 2D del género survival horror para PC, el objetivo es matar el maximo numero de muertos vivientes. Este es un proyecto final para la materia Programación Orientada a Objetos y se encuentra finalizado, se desarrollo con C++ y la librería SFML, sin utilizar motores gráficos.

CRUD

CRUD basico donde podemos agregar, editar y borrar nombres, ademas la lista de nombres se ve a un costado, se creo con ReactJS, Uniqid y Bootstrap.

Calculo 1

El fin de este proyecto es digitalizar los contenidos de la asignatura Calculo 1 (FICH, UNL), su contenido consiste en notas, teoremas, criterios y apuntes más influyentes en el estudio de la materia.

Simulador

Este proyecto esta siendo desarrollado en el Lenguaje NetLogo. Mi rol es el de investigar sobre la calidad del ambiente socio-físico-simbólico de un lugar y las características socio-demográfica de su población, para luego plasmarlo en un simulador.
